# phan-tich-du-lieu-so-sanh-mc-mu
Comparing Manchester City and Manchester United during period 2007 to 2023
This is a file code comparing Manchester City and Manchester United throughout 2007 to 2023 (as the peak season of MU and MC during this period)
By comparing match features, compact summary, average points, peak phase and elo,... I have made some comparisons between these two teams to pull out some insights and decide whether which team was stronger
